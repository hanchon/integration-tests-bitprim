# Integration tests bitprim
WIP use python to generate integration tests for a bitprim node.

# Requirements
* Python 3.X
* Bitprim-exe with rpc.
```
cd integration-tests-bitprim
cd scripts
./download_bitprim.sh
```

# Run
```
git clone https://github.com/hanchon/integration-tests-bitprim
cd integration-tests-bitprim
cd scripts
python3 run.py
```
